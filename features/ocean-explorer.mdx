The Ocean Explorer page (`/landing`) is a continuous scroll journey through Thai waters. As you scroll down, each of the 10 endangered marine animals glides into view alongside an info box. The depth indicator tracks your position from 0 m at the surface to 2000 m at the ocean floor.

## Page structure

The page is built from a stack of `<section>` elements, each dedicated to one animal or transitional scene. All animal visuals are rendered as polygon shard art using the `ANIMALS` data from `animalConfig.js`. Every section uses the `ParallaxElement` component to drive motion.

<Note>
See [Parallax](/features/parallax) for full documentation on `ParallaxElement` props and behavior.
</Note>

## AnimalInfoBox

Each animal's info box is rendered by the `AnimalInfoBox` component. It reads from `animalDetails.json` using the `animalId` prop, then renders:

- The English name and Thai name side by side
- The `content.intro` text from the animal's detail record
- A "รายละเอียด" link button that navigates to `/detail?id={animalId}`

The box itself is wrapped in a `ParallaxElement` with `yRange: ["20%", "0%", "-5%"]` and `opacityRange: [0, 1, 1]`, so it fades in and rises as you scroll into its section.

```jsx
<AnimalInfoBox
  animalId="irrawaddyDolphin"
  align="left"       // "left" | "right"
  isOpaque={false}   // adds .opaque-v2 class for a solid background
  className="irrawaddy-info-v2"
/>
```

## Ambient scene components

### LightRays

Five CSS-animated diagonal beams (`.light-ray-v2.r1` through `.r5`) simulate sunlight penetrating from the surface. These are only rendered in the hero section.

### MarineSnow

Thirty floating particles are generated with `useMemo` at mount time, each with a random size (2–6 px), random position, and random animation duration (20–35 s) and delay (0–10 s). They drift continuously across the full-page background, adding depth and motion.

### DepthIndicator

A fixed ruler on the left edge of the viewport shows depth marks at 0 m, 500 m, 1000 m, 1500 m, and 2000 m. The current depth value is derived from `scrollYProgress` mapped to 0–2000 and smoothed with a spring (`stiffness: 100, damping: 30`).

```js
const depth = useTransform(scrollYProgress, [0, 1], [0, 2000]);
const roundedDepth = useSpring(depth, { stiffness: 100, damping: 30 });
```

## Background and audio

The `UnderwaterEnvironment` component renders behind all content as a 3D underwater scene. Background music starts automatically on first user interaction via `SoundController`. Both components are imported from their respective files:

```jsx
import UnderwaterEnvironment from "../components/UnderwaterEnvironment";
import SoundController from "../components/SoundController";
import bgSound from "../assets/sound/bg-sound.MP3";

// Usage:
<UnderwaterEnvironment />
<SoundController bgSound={bgSound} />
```
