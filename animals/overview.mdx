Low-Poly Ocean features ten endangered marine species found in Thai waters. Each animal is rendered as low-poly polygon art — a mosaic of triangular shards that animates as you explore the experience.

The animals were chosen because they are all present in Thai seas and all face real conservation pressures: bycatch, plastic pollution, coastal development, and illegal trade. Several are listed on the IUCN Red List and protected under Thai law.

## Bilingual content system

All educational content in the experience is stored in `src/data/animalDetails.json`. Each animal entry holds three topics:

| Key | Thai label | Description |
|---|---|---|
| `intro` | ทำความรู้จัก | Introduction and physical description |
| `threat` | ภัยคุกคาม | Threats to the species |
| `solution` | แนวทางการแก้ไข | Conservation solutions |

The content is authored in Thai, which is the primary language of the experience. The documentation pages in this section summarize the key facts in English for developers and contributors who are not Thai speakers.

## Polygon shard counts

Each animal artwork is built from a fixed set of triangular polygon shards defined in `src/data/animalConfig.js`. The blue whale has the most shards (433), making it the most visually complex artwork. The irrawaddy dolphin has the most shards (125) among the cetaceans with a body-length silhouette.

<Note>
  Shard counts reflect the number of `clipPath` polygon entries in the `shards[]` array for each animal in `animalConfig.js`. Shard count directly affects the visual complexity and animation weight of each artwork.
</Note>
