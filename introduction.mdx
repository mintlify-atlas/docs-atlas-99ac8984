## What is Low-Poly Ocean?

Low-Poly Ocean is an immersive, scroll-driven web experience built with React, Three.js, and Framer Motion. It takes users on a visual journey through the ocean to meet 10 endangered or threatened marine species found in Thai waters — each rendered as a beautiful, animated low-poly polygon artwork.

The project blends conservation awareness with digital art, using geometric polygon shards to represent each animal. The fragmented, reconstructable nature of the artwork mirrors the fragility of marine ecosystems: if one piece is missing, the whole picture falls apart.

## Key Features

<CardGroup cols={2}>
  <Card title="Low-Poly Animal Art" icon="shapes">
    Each animal is built from dozens of CSS polygon shards with precise clip-path coordinates and colors sampled from real photographs.
  </Card>
  <Card title="Parallax Scroll Experience" icon="layers">
    Smooth spring-based parallax animations powered by Framer Motion's `useScroll` and `useSpring` hooks create an immersive underwater descent.
  </Card>
  <Card title="3D Underwater Environment" icon="water">
    A full 3D ocean scene built with React Three Fiber (@react-three/fiber) and Three.js renders in the background.
  </Card>
  <Card title="Three-Stage Journey" icon="map">
    Users progress through three stages: Ocean Explorer → Animal Details → Why Polygon Art?
  </Card>
  <Card title="Bilingual Content" icon="globe">
    All animal information is available in both English and Thai, covering introduction, threats, and conservation solutions.
  </Card>
  <Card title="Ambient Sound" icon="music">
    An ambient ocean soundscape plays in the background with a user-controlled sound toggle.
  </Card>
</CardGroup>

## The 10 Featured Animals

All animals featured in Low-Poly Ocean are endangered or threatened species found in Thai waters:

| Animal | Thai Name | Status |
|--------|-----------|--------|
| Leatherback Turtle | เต่ามะเฟือง | Protected Wildlife |
| Dugong | พะยูน | Protected Wildlife |
| Irrawaddy Dolphin | โลมาอิรวดี | Critically Endangered |
| Pink Dolphin (Indo-Pacific Humpback) | โลมาชมพู / โลมาหลังโหนก | Vulnerable |
| Stingray | ปลากระเบน | Various |
| Orca (Killer Whale) | วาฬเพชฌฆาต | Data Deficient |
| Sperm Whale | วาฬสเปิร์ม | Vulnerable |
| Blue Whale | วาฬสีน้ำเงิน | Protected Wildlife (Thailand) |
| Sawfishes | ปลาฉลามเลื่อย | Critically Endangered |
| Ronin (Manta Ray) | ปลากระเบนปีศาจ | Vulnerable |
